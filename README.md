### CLI

```bash
USAGE:
   halftone [global options] command [command options] [arguments...]

COMMANDS:
   help, h  Shows a list of commands or help for one command

GLOBAL OPTIONS:
   --output value, -o value  output file name (default: "output%s")
   --help, -h                show help
   --version, -v             print the version

EXAMPLE:
    halftone -o output.png input.png
```
